# Embeejayz Coleccionables

Este proyecto busca centralizar todas las aplicaciones del ecosistema de recompensas para **Embeejayz**.

Cómo podrá observar el ecosistema es grande para lo que podrías imaginarte, entre las apps se incluyen:

- Gountz Bot (Bot para twitch, canjea recompensas e interactua de forma divertida con los usuarios)
- Web Coleccionables (UI para que los usuarios vean sus recompensas)
- Overlay Notifications (UI para que Embee la integre en obs, para notificar a los usuarios que canjearon sus recompensas)

Si bien esto es un monorepo, cada aplicación tiene su propia documentación por su complejidad

Documentaciones:

[GountzBot](./docs/gountzbot.md)

## Tecnologías del monorepo

- NX
- pnpm
- Docker
- Github actions

## Instalación

```sh
pnpm install
```
